# Support

Use GitHub Issues for:

- bugs
- setup problems
- documentation gaps
- feature requests
- API contract questions

Before opening an issue, please include:

- operating system
- Python version
- Node.js version
- whether `LEGALEAGLE_DEMO_MODE=true` was used
- backend logs around the failure
- frontend console/network errors, if relevant

For legal questions, consult a qualified legal professional. LegalEagle is an informational research tool and cannot provide legal advice.
