# Roadmap

## MVP Stabilization

- Confirm local availability of model weights.
- Keep demo mode reliable for development.
- Keep `/chat` response shape aligned with the frontend.
- Improve error messages for missing models and failed uploads.
- Add basic backend and frontend smoke tests.

## Source Quality

- Add structured metadata during document ingestion.
- Preserve citations, court, jurisdiction, year, and document type.
- Display richer source cards in the frontend.
- Add source filtering by legal category.

## Product Features

- Saved chat threads
- Server-side conversation persistence
- Export answers and citations
- Document management page backed by real uploads
- Better confidence scoring
- Searchable source library

## Platform and Security

- Authentication
- Restricted CORS
- Per-user document indexes
- Rate limiting
- Production deployment guide
- Observability and logging

## Future Enhancements

- SSE streaming via `/chat/stream`
- Multi-language support
- More legal domains and datasets
- Evaluation set for legal QA quality
- Citation verification workflow
