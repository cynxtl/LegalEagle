# ⚖️ LegalEagle — AI-Powered Legal Chatbot (RAG-Based)

## 🚀 Overview
LegalEagle is an AI-powered legal assistant designed to make Indian legal information accessible, understandable, and interactive for everyone.

It uses a Retrieval-Augmented Generation (RAG) architecture combining:
- Legal knowledge base
- Semantic search (FAISS + embeddings)
- Large language model (LLM)

---

## 🎯 Problem Statement
Access to legal knowledge in India is limited due to:
- Low legal awareness
- Complex legal language
- Limited access to lawyers
- Court backlog
- Lack of interactive tools

---

## 💡 Core Idea
LegalEagle retrieves relevant legal documents and generates grounded responses to avoid hallucinations.

---

## 🧠 System Architecture
User Query → Embedding → FAISS → Relevant Docs → LLM → Response
